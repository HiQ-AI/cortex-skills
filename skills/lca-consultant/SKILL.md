# LCA 搜索咨询策略

> **重要**：本技能中的 GWP 数值用于内部判断（是否需要引导）和专业点评（用户问"合不合理"时作参考）。但**不要在数据集结果表格里填入这些值** — 表格里的 GWP 只能来自搜索工具返回。

## 核心框架：五维度判断

收到一个搜索请求时，按以下五个维度快速判断哪些需要澄清、哪些用默认值：

```
用户查询 → [1.技术路线] → [2.地理] → [3.功能单位] → [4.时间] → [5.系统模型] → 搜索
              ↓需要问吗？    ↓需要问吗？   ↓匹配吗？      ↓用默认     ↓用默认
```

**原则：最多问 1-2 个维度。** 技术路线和地理是最常需要澄清的，其余用默认值。

---

## 维度 1：技术路线

**最关键的维度。** 同一材料不同技术路线 GWP 可差 2-10 倍。

### 判断规则
- 用户指定了具体工艺 → 不问，直接搜
- 该材料只有一条主流路线 → 不问，直接搜
- 该材料有多条路线且 GWP 差异 >2x → 引导

### 钢铁

| 用户说 | 行业术语 | 工艺含义 | 搜索词 | GWP |
|--------|---------|---------|--------|-----|
| 长流程/转炉钢 | BF-BOF | 铁矿石→高炉→转炉→粗钢 | blast furnace, basic oxygen furnace | 1.7-2.5 |
| 短流程/电炉钢 | EAF | 废钢→电弧炉→粗钢 | electric arc furnace, steel scrap | 0.4-0.8 |
| 半长流程 | DRI+EAF | 天然气直接还原+电弧炉 | direct reduced iron, electric arc furnace | 1.2-1.8 |
| 绿氢直还 | H-DRI+EAF | 绿氢还原铁+电弧炉（前沿） | hydrogen-based DRI, green steel | 0.3-0.6 |
| 热轧 | 热轧卷/板/型材 | 粗钢→加热→轧制 | hot rolling, steel coil/sheet/plate | 粗钢+0.1-0.3 |
| 冷轧 | 冷轧板/带 | 热轧→酸洗→冷轧 | cold rolling, steel sheet | 热轧+0.1-0.2 |
| 镀锌 | 热镀锌/电镀锌 | 冷轧板→镀锌 | galvanized steel, zinc coating | 冷轧+0.3-0.5 |
| 螺纹钢/钢筋 | 热轧棒材 | 长流程→热轧 | reinforcing steel, steel rebar | 2.0-2.5 |
| 不锈钢 | 304/316/430 | 电弧炉+AOD精炼 | stainless steel, chromium steel 18/8 | 4.5-6.5 |
| 球团/烧结矿 | 炼铁原料 | 铁矿粉造球/烧结 | iron ore pellet / sinter | 0.1-0.3 |

中国 ~90% 长流程。用户不指定时默认按长流程搜。

### 铝

| 用户说 | 工艺含义 | 搜索词 | GWP |
|--------|---------|--------|-----|
| 原铝/电解铝 | 氧化铝→电解 | primary aluminium, electrolysis | CN:14-20, EU:8-10 |
| 再生铝/二次铝 | 废铝→重熔 | secondary aluminium, scrap | 0.5-1.5 |
| 铝型材 | 挤压成型 | aluminium extrusion, profile | 基础铝+0.5-1.0 |
| 铝板/铝卷 | 轧制 | aluminium sheet, rolling | 基础铝+0.3-0.5 |
| 铝箔 | 极薄轧制(6-200μm) | aluminium foil | 基础铝+1.0-2.0 |
| 压铸铝 | ADC12等铸造合金 | aluminium die casting | 多用再生铝+铸造 |

原铝 vs 再生铝差 4-10 倍，**必须问**。

### 塑料

| 用户说 | 需要进一步区分 | 搜索词 | GWP |
|--------|--------------|--------|-----|
| PE/聚乙烯 | HDPE vs LDPE vs LLDPE | polyethylene, high/low density | 1.8-2.5 |
| PP/聚丙烯 | — (直接搜) | polypropylene, granulate | 1.5-2.5 |
| PVC/聚氯乙烯 | 硬质(管材) vs 软质(膜) | PVC suspension / PVC emulsion | 1.8-2.8 |
| PET | 瓶级 vs 纤维级 | PET bottle grade / PET fibre | 2.5-3.5 |
| ABS | — (直接搜) | acrylonitrile butadiene styrene | 3.0-4.0 |
| PC/聚碳酸酯 | — (直接搜) | polycarbonate | 5-7 |
| PA/尼龙 | PA6 vs PA66 | polyamide 6 / polyamide 6.6 | 6-9 |
| 改性/GF | 基材+玻纤比例 | glass fibre reinforced + 基材名 | 基材+10-30% |
| 再生塑料 | 哪种塑料的再生 | recycled + 具体塑料名 | 原生的30-70% |

用户说"塑料"必须问是哪种。说了具体名称(PP/ABS等)直接搜。

### 化工

| 用户说 | 歧义 | 关键区分 |
|--------|------|---------|
| NaOH/烧碱 | 氯碱联产，分配方法影响巨大 | 膜法 vs 隔膜法，分配方式 |
| 乙醇 | 石化法 vs 发酵法 = 2-3x | 合成 vs 生物基 |
| 氢气 | 灰氢 vs 蓝氢 vs 绿氢 = 5-10x | 天然气制氢 / 电解水 |
| 氨 | 天然气合成 vs 绿氨 = 3-5x | 常规 vs 绿色 |
| "助剂"/"添加剂" | 完全不知道是什么 | 必须问具体化学名或CAS号 |

### 能源

| 用户说 | 搜索词 | 关键变量 |
|--------|--------|---------|
| 电力/市电/网电 | electricity, grid mix | **地区**（见维度2） |
| 绿电 | electricity, wind/solar/hydro | 具体可再生类型 |
| 蒸汽 | steam, [pressure], from [fuel] | 压力等级 + 燃料类型 |
| 天然气(燃烧) | natural gas, burned in | 直接搜 |
| 天然气(原料) | natural gas, at plant | 上游开采 |
| 柴油(燃烧) | diesel, burned in | 直接搜 |

### 运输

| 用户说 | 需要区分 | 搜索词 |
|--------|---------|--------|
| 运输/物流 | 必须问运输方式 | — |
| 海运 | 集装箱 vs 散货 | container ship / bulk carrier |
| 公路/陆运 | 卡车吨位 | lorry, 16-32t / 3.5-7.5t |
| 铁路 | 电力 vs 内燃 | freight train, electric/diesel |
| 空运 | 客机腹舱 vs 货机 | air freight, long-haul |

## 维度 2：地理/地区

### 判断规则
- 用户指定了地区 → 不问
- 材料对地区**高敏感** → 引导选择
- 材料对地区**低敏感** → 用 GLO 或不限

### 地区敏感度分级

| 敏感度 | 材料 | 原因 | 处理 |
|--------|------|------|------|
| 🔴 极高 | 电力、电解铝 | 电网结构决定一切 | **必须问** |
| 🟠 高 | 钢铁、水泥 | 工艺结构和能源组合不同 | 建议问 |
| 🟡 中 | 玻璃、纸 | 能源组合有差异 | 可以问 |
| 🟢 低 | 大宗塑料、基础化工品 | 全球石化路线类似 | 不用问，GLO 可用 |

### 常用地区选项（HITL 按钮）
- 中国 / CN
- 全球平均 / GLO
- 欧洲 / RER
- 不限（搜所有）

## 维度 3：功能单位

### 判断规则
- 用户和数据库单位一致 → 不提
- 单位不匹配但可换算 → 搜完提醒换算关系
- 单位不匹配且无法换算 → 搜前提醒

### 常见单位错配

| 用户期望 | 数据库常用 | 换算提示 |
|---------|-----------|---------|
| 每米管材 | per kg | 需要知道线密度(kg/m) |
| 每平方米板材 | per kg | 需要知道面密度(kg/m²) = 密度×厚度 |
| 每个零件 | per kg | 需要知道单件重量 |
| 每 kWh 电池 | per kg | 需要知道能量密度(kWh/kg) |
| 每 tkm 运输 | per kg·km | 1 tkm = 1000 kg·km |
| 每升液体 | per kg | 需要知道密度 |

搜索时始终用数据库原始单位(通常 per kg)。在结果里标注：如果用户需要其他单位，告诉他怎么换算。

## 维度 4：时间代表性

### 判断规则
- **不主动问。** 默认搜最新版本。
- 以下场景需要提醒用户注意时效性：

| 材料 | 时间敏感原因 | 提醒内容 |
|------|-------------|---------|
| 电力 | 电网结构逐年变化（可再生占比增长） | "数据库电力数据可能滞后 2-3 年，实际电网碳强度可能更低" |
| 锂电池 | 制造效率每年改善 10-15% | "近两年电池数据可能已有显著改进" |
| 太阳能板 | 同上 | "光伏制造能耗持续下降" |
| 氢气(绿氢) | 技术快速发展 | "绿氢数据变化快，注意数据年份" |

**成熟工业品（钢、铝、水泥、塑料等）5 年内数据均可接受。**

---

## 维度 5：系统模型

### 判断规则
- **默认 cut-off（APOS）。** 不主动问。
- 以下情况需要切换或提醒：

| 信号 | 建议模型 | 原因 |
|------|---------|------|
| 用户提到"consequential" | consequential | 用户明确要求 |
| 用户做决策对比（"换成材料 B 会怎样"） | consequential | 边际效应分析 |
| 高回收含量材料（再生铝、再生塑料） | 提醒两种模型差异大 | cut-off 中回收料无负担，consequential 中有市场效应 |
| 联产品（氯碱、炼油） | 提醒分配方式 | 不同模型分配结果差异大 |
| 产品碳足迹 / ISO 14067 | cut-off | 行业惯例 |
| 欧盟 PEF | CFF (Circular Footprint Formula) | 欧盟法规要求 |

## 政策法规对数据选择的约束

用户提到以下法规/标准时，搜索策略需要调整：

### 欧盟 CBAM（碳边境调节机制）
- **适用产品**（2026 定义阶段）：钢铁、铝、水泥、化肥、电力、氢
- **2028 扩围提案**：下游金属制品（机械零部件、家电、建筑设备等含钢/铝 >95% 的产品）
- **2027 评估**：化工行业可能纳入
- **数据要求**：优先用实际生产数据（前景数据），背景数据用 CBAM 默认值或同等质量数据集
- **系统边界**：直接排放（Scope 1）+ 间接排放（Scope 2 电力），不含 Scope 3
- **搜索影响**：搜到数据集后要确认是否包含 Scope 1+2，排除 cradle-to-gate 全生命周期数据

### 欧盟 PEF（产品环境足迹）
- **系统模型**：必须用 CFF（Circular Footprint Formula），不是 cut-off 也不是 consequential
- **数据源**：优先 EF 数据库（Environmental Footprints），其次 Ecoinvent
- **搜索影响**：设 sources 过滤为 EF 或 Ecoinvent，提醒用户 CFF 分配方式

### ISO 14067（产品碳足迹）
- **数据质量**：要求数据不超过 5 年（时间代表性）、地区匹配（地理代表性）、技术匹配
- **系统模型**：通常 cut-off，但回收处理可选择其他方法
- **搜索影响**：优先最新版数据库，优先匹配地区

### ISO 14044 / 14040（LCA 通用框架）
- **分配**：避免分配 > 系统扩展 > 物理关系分配 > 经济分配
- **截断规则**：质量/能量/环境显著性 <1% 的输入可忽略
- **搜索影响**：对质量占比 <1% 的辅料可建议用通用代理数据

### GHG Protocol / Scope 3
- **15 个类别**各有推荐数据源和计算方法
- **关键类别**：购买的商品和服务（Cat 1）、运输（Cat 4/9）、使用阶段（Cat 11）
- **搜索影响**：要区分 cradle-to-gate（Cat 1 用）和 well-to-wheel（Cat 4 运输用）

### 中国碳足迹相关
- **碳足迹管理体系（2025）**：生态环境部发布 45 项国标计划，覆盖锂电池、新能源车、电子产品碳足迹核算
- **碳标签认证**：产品碳标签认证体系建设中，预计分批发布产品种类规则
- **碳市场扩围**：2027 前扩展到化工、石化、民航、造纸
- **"十五五"（2026-2030）**：碳排放双控，强度为主、总量为辅
- **搜索影响**：中国法规场景优先用 HiQLCD（中国本地数据），其次 TianGong

### 判断规则
- 用户提到法规名称 → 自动应用对应约束
- 用户没提法规但提到"合规"、"报告"、"申报" → 问是哪个法规
- 用户没提任何法规 → 不主动问，用通用方法

## 产品→材料拆解

用户说产品名时，先解释组成再决定搜什么。

### 常见产品拆解

**排水管**：PVC 管体 90-95% + 碳酸钙填料 3-5% + 稳定剂 2-3%
→ 数据库有 PVC pipe extrusion 完整数据集，直接搜

**电缆**：铜导体 + XLPE/PVC 绝缘 + 钢带铠装 + PVC 护套
→ 4 种材料分别搜，按截面积估算各材料重量

**瓦楞纸箱**：面纸(牛皮/再生) 55-65% + 瓦楞芯纸 35-45%
→ 数据库有 corrugated board 聚合数据集

**锂电池包**：正极(NMC/LFP) 25-35% + 负极石墨 12-15% + 电解液 8-12% + 铜箔/铝箔 + 壳体
→ 先问化学体系（NMC vs LFP），再决定搜组件还是聚合数据集

**太阳能板**：玻璃 65-70% + 硅电池 3-5% + EVA 5-7% + 铝框 10-15%
→ Ecoinvent 有聚合数据集 photovoltaic panel production

**LED 灯泡**：铝散热器 30-40% + PC 灯罩 10-15% + PCB 驱动 10-15% + LED 芯片 <5%
→ 按组件分别搜

**T 恤(棉)**：棉纤维 85-92% + 涤纶缝线 1-2% + 染整化学品 3-5%
→ 湿法加工(染整)占总碳足迹 30-50%，不能跳过

拆解知识不够时，用 web search 查产品组成。

## 引导交互方式

### 用 AskUserQuestion 给按钮选项

规则：
1. **先说差异，再给选项**。告诉用户为什么需要区分。
2. **必须有"都搜/不确定"选项**。不强迫用户选。
3. **一次最多 2 个字段**。
4. **像同事建议，不像表单填写**。
5. **给专业建议**。"如果不确定，建议按长流程搜——中国 90% 产能是长流程。"

### 不需要引导的情况

直接交给搜索：
- 具体材料名（304不锈钢、HDPE、PA66-GF30）
- 具体工艺（电弧炉、注塑成型、克拉夫法）
- 数据库/标准引用（Ecoinvent 3.12、cut-off）
- CAS 号、合金牌号、化学式
- 用户说"先搜搜看"、"都搜出来我对比"

## 与 lca-search 的配合

1. 本技能解决**搜什么**（五维度判断、术语映射、产品拆解）
2. lca-search 解决**怎么搜**（query 构建、同义词、结果评估）
3. 引导完成后，把明确的材料名+参数交给搜索工具
